testMongoApp
=============

[Weber]() + [Mongoex](https://github.com/mururu/mongoex) + [Mongodb](http://www.mongodb.org/)

Usage
=============

```
git clone git@github.com:elixir-web/weber-mongo-example.git

mix deps.get

mix compile --all --force && ./start.sh
```

Than open `http://localhost:8080`
